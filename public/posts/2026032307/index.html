    四、 如何用人民币购买加密货币？ #

    在币安，我们需要先用人民币购买 USDT（一种价格与美元 1:1 锚定的稳定币，类似于充值游戏里的 Q币），然后再用 USDT 去购买比特币等其他加密货币。

    C2C 买币步骤： #

    1. 打开币安 APP 首页，点击 “C2C 买币”。
    2. 页面分为“快捷区”、“自选区”和“大宗区”。快捷区汇率可能略高，推荐在自选区挑选商家。
    3. 筛选商家（关键所在）： * 可以通过支付宝、微信或银行卡筛选支付方式。
      • 防冻卡秘籍： 一定要寻找名字旁边带有**“盾牌”标志的商家，并且要看他们的交易量高、完单率高**。这类商家的资金相对干净安全，出入金不容易导致银行卡被冻结（虽然价格可能会稍微贵一点点，但安全第一）。
    4. 选好商家后，点击“买入”，按照平台提示向商家付款，付款后点击“我已付款”。
    5. 商家确认收款后，平台会自动将 USDT 发放到你的币安资金账户中，全程类似淘宝购物，非常安全。

    五、 资金划转与开始交易 #

    买到 USDT 后，钱默认在“资金账户”里，需要划转后才能交易。

    1. 资金划转： 点击右下角“资产” -> “划转”。将 USDT 从**“资金账户”划转到“现货账户”**（如果想玩合约，就划转到“合约账户”），点击确认。
    2. 购买比特币： 回到首页，搜索“BTC”（比特币），点击进入交易页面。输入你想购买的数量或金额，拖动进度条，点击“买入”即可。卖出操作同理。

    六、 提现与转账（新手必看避坑指南） #

    如果你有多个交易所，或者想把 USDT 转到支持加密货币入金的海外券商炒股，就需要用到“提币”功能。

    1. 点击“资产” -> “提现”，选择你想提取的币种（如 USDT）。
    2. 粘贴收款地址并选择网络： 这里是重中之重！你选择的转账网络（例如 BEP20、TRC20、ERC20）必须与对方收款地址的网络完全一致！
    3. ⚠️ 严重警告： 网络如果选错，转出去的钱将会永久丢失，无法找回！操作前务必反复核对。
    4. 确认数量和网络无误后，点击提交，输入验证码，资金就会转移到目标地址。
