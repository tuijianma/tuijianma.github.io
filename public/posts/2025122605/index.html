    三、币安身份认证（KYC）流程说明 #

    1️⃣ 选择国家或地区 #

    中国大陆用户在地区选项中请选择【中国】。

    2️⃣ 填写真实信息 #

    根据页面提示填写个人信息，并上传有效身份证件。

    3️⃣ 提交审核 #

    提交后系统会自动审核，通常速度较快。审核通过后，即可解锁 C2C 买币、提高交易额度等功能。

    四、如何在币安进行买卖交易 #

    1️⃣ 进入交易页面 #

    在 App 底部点击【交易】或【市场】，进入交易界面。

    2️⃣ 选择交易对 #

    在搜索栏中选择想交易的币种，例如 BTC/USDT，表示使用 USDT 买入或卖出比特币。

    3️⃣ 选择下单方式 #

    常见订单类型包括：

    • 市价单：按当前市场价格立即成交
    • 限价单：自行设定买入或卖出价格，等待市场触发

    4️⃣ 下单操作 #

    输入价格和数量后：

    • 买入点击【买入】

    5️⃣ 查看成交记录 #
