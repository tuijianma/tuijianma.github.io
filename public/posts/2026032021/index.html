    第二步：下载币安官方APP #

    第三步：身份认证与安全设置（资金护盾） #

    登录APP后，为了账户能正常使用且保障资金安全，必须完成以下两步。

    1. 身份认证（KYC） #

    • 登录后系统会自动引导认证，居住地选择中国。
    • 直接使用中国身份证即可，按提示上传正反面照片并完成人脸识别。
    • 审核通常非常快，通过后即可解除交易限制。

    2. 双重安全验证（必做！） #

    币圈最重要的法则就是安全。强烈建议开启以下两个功能：

    • 通行密钥（Passkeys）： 在“账户安全”中开启，绑定你的人脸信息（FaceID）或指纹。这会将密钥保存在你的设备或iCloud中，是目前最安全的验证方式。
    • 谷歌验证器（Google Authenticator）：
      1. 在APP Store或Google Play下载“Google Authenticator”。
      2. 在币安“安全”设置中选择开启验证器，复制币安给出的一串密钥。
      3. 打开谷歌验证器，点击“+”号 -> “输入设置密钥”，粘贴密钥并保存。
      4. 以后登录或提现时，都需要输入谷歌验证器上生成的6位动态验证码（类似于银行的U盾）。

    第四步：如何入金（购买USDT） #

    在币圈，USDT（泰达币）相当于“美元筹码”，我们需要先用人民币买入USDT，再用USDT去购买比特币等其他币种。

    1. 进入C2C交易： 在首页点击“充值”或“C2C交易”。
    2. 筛选商家（关键）：
      • 不要只看价格： 也就是不要直接点“快捷买币”，因为系统自动匹配的低价商家可能存在风险。
      • 进入自选区： 点击右上角漏斗筛选，支付方式选“微信”或“银行卡”，并勾选**“神盾商家”**（这类商家缴纳了高额保证金，更安全）。
    3. 挑选靠谱商家：
      • 看成交量：越高越好。
      • 看成单率：最好在98%以上。
      • 看注册时间：优先选择注册时间超过1-2年的老商家。
    4. 下单付款： 输入金额下单。切记：
      • 使用本人实名账户转账（不要代付）。
      • 转账备注里不要填写任何“比特币、BTC、USDT、币安”等敏感字眼，防止银行卡风控。
      • 付款后截图保存，回到币安点击“我已付款，通知卖家”。

    第五步：现货交易（买入比特币） #

    买到USDT后，它默认在“资金账户”。交易前需先进行“划转”。

    1. 资金划转： 点击“资产” -> “划转”，将USDT从资金账户划转到现货账户（此过程免费）。
    2. 开始交易： 点击底部“交易”按钮，进入现货交易页面。
    3. 看懂界面：
      • 交易对： 例如想买比特币，搜索并选择 BTC/USDT。
      • 买盘/卖盘： 红色代表卖家报价，绿色代表买家报价。
    4. 下单方式：
      • 限价单（Limit）： 设定一个你心仪的低价（比如BTC跌到83000时买入），挂单等待成交。
      • 市价单（Market）： 不在乎价格，按当前市场最优价格立即买入（适合急于上车）。
    5. 止盈止损： 为了控制风险，可以使用“止盈止损”单。设定触发价格，当币价跌破某条线（如77000）时自动卖出，防止亏损扩大。

    第六步：出金（变现与防冻卡技巧） #

    当你赚了钱想换回人民币时，操作与入金相反，但风险更高（主要是防止收到涉诈黑钱导致银行卡冻结）。

    防冻卡核心原则：

    1. 专卡专用： 绝对不要用工资卡、房贷卡或存有大量积蓄的主卡收款。
    2. 降低频率： 不要频繁出金，建议积累一定金额后少次提取。
    3. 严选买家：
      • 在C2C卖出区，同样筛选“神盾商家”。
      • 查看买家详情：注册时间要长，好评率要高。
      • 警惕异常报价： 如果某个商家收币价格远高于市场价（例如市场价6.7，他出6.84），千万别卖给他，这极大概率是急于洗钱的黑资。
    4. 实名一致性： 收到钱后，检查付款人姓名是否与币安显示的实名一致。如果不一致（例如用家属账号转账），立即原路退回并取消订单，这100%是陷阱。
