    想买黄金却不想跑金店，又觉得支付宝黄金 ETF 限制太多？作为一名有 3 年经验的加密玩家，我曾深度持有过 XAUT 和 PAXG。今天就把最简单的操作逻辑和“避坑指南”分享给大家。

    什么是黄金 RWA？ 简单说就是把实物黄金“上链”。每持有 1 个代币，就代表在银行金库里锁死了一盎司实物黄金。

    一、 XAUT vs PAXG：我该选哪一个？ #

    这两个是目前市面上最靠谱的黄金代币，但它们的“性格”不太一样：

    特性 XAUT (Tether Gold) PAXG (Paxos Gold)
    发行商 Tether (USDT 的母公司) Paxos (受纽约州监管)
    背书实力 极强，信誉与 USDT 深度绑定 极高，拥有美国合规信牌照
    主要战场 欧易 OKX、Bybit、Bitget 币安 Binance
    适合人群 追求高流动性、USDT 忠实用户 追求合规透明、币安老用户

    经验谈：如果你追求交易丝滑，推荐 XAUT；如果你非常看重监管背书，选 PAXG。

    二、 保姆级购买教程（以欧易 OKX 买 XAUT 为例） #

    第一步：账号注册与手续费减免 #

    打新和买金，手续费积少成多。通过下方专属链接注册，可直接减免交易手续费：

    • 欧易 OKX 注册（自动减免 20%）
    • 币安 Binance 注册（现货减 35%，合约减 25%）

    第二步：法币换 USDT (C2C) #

    1. 在首页点击 [买币] -> [C2C 交易]。
    2. 选择信誉好的商家，建议勾选“神盾”或“新手友好”标识。
    3. 支付完成后，USDT 会进入你的【资金账户】，记得划转到【交易账户】。

    第三步：现货买入 XAUT（避坑关键！） #

    1. 点击底部的 [交易] -> 选择 [现货]。
    2. 搜索 XAUT/USDT 交易对。
    3. 核心秘籍：千万别点“闪兑”！闪兑汇率坑很大。请直接用现货限价或市价买入。
      • 现货手续费通常只要 0.08%，而闪兑的隐形成本可能高达 0.5%。

    三、 进阶玩法：黄金也能“生利息”？ #

    实物金条放在家里是不会变多的，但在 Web3 世界可以：

    1. 质押借贷：在交易所把 XAUT 当作抵押品，借出 USDT 去参加新币挖矿（Launchpool），实现“一份资产，两份收益”。
    2. 活期理财：目前 Bitget 交易所 的黄金代币理财相对给力，常有 2-3% 的年化，比干拿现货强。
    3. 反向做空：如果你觉得金价到顶了，可以在 Bybit 或欧易开 1-3 倍杠杆做空 XAUT。

    四、 为什么放弃实物金条改买 RWA？ #

    1. 门槛极低：实物金条动辄几千上万，XAUT 10 美元起买。
    2. 交易自由：7×24 小时随买随卖，不受节假日限制。
    3. 变现丝滑：换回 USDT 后，在全球范围内都有极佳的流动性，无需像实物金回收那样被压价。
    4. 存储安全：不用担心假货，也不用买保险柜，资产在区块链上由全球金库承保。
